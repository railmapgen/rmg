# Rail Map Generator (RMG)

A powerful tool for generating railway maps and information panels in the styles of major public transportation systems:

- **Guangzhou Metro** (Guangzhou)
- **MTR** (Hong Kong)
- **Shanghai Metro** (Shanghai)
